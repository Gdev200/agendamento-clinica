# Sistema de Agendamento Clínico

Este projeto é um sistema web desenvolvido como parte dos meus estudos na faculdade de [Nome do Seu Curso], focado na solução de agendamento de consultas para uma clínica.

## 🚀 Funcionalidades Principais

*   **Agendamento de Consultas:** Interface para cadastro de data, horário, paciente e especialidade.
*   **Gestão de Horários:** Lógica de validação para evitar conflitos de horários e datas inválidas.
*   **Persistência de Dados:** Modelagem eficiente em SQL para armazenamento seguro das informações.

## 🛠️ Tecnologias Utilizadas

*   **Front-end:** HTML5, CSS3, Bootstrap (para responsividade).
*   **Back-end:** PHP (lógica de negócio e validações).
*   **Banco de Dados:** MySQL (modelagem de dados).
*   **Modelagem (Arquitetura):** Conceitos de UML utilizados para o desenho dos fluxos de agendamento.
